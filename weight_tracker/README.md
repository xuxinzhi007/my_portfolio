# 体重记录功能 - 增强版

这是一个使用 Fyne 框架实现的体重记录工具，可以帮助你追踪体重变化，带有美观的界面和动画效果。

## ✨ 功能特性

### 📊 统计面板
- **当前体重** - 显示最新的体重记录
- **总变化** - 显示从第一条记录到现在的总体变化（增加/减少）
- **记录数量** - 显示总共记录的次数
- **最高/最低体重** - 显示历史最高和最低体重

### 📝 记录管理
- ✅ 记录体重数据（支持小数，如 70.5）
- ✅ 自动计算每次体重变化（增加/减少/持平）
- ✅ 显示历史记录列表（按时间倒序）
- ✅ 数据持久化（JSON 文件存储）
- ✅ 输入验证（范围：20-300 kg）

### 🎨 界面优化
- 🎯 卡片式设计，信息层次清晰
- 🌈 彩色标识，不同状态一目了然
- ✨ 添加记录时的淡入淡出动画
- 📱 响应式布局，适配不同窗口大小
- 🎭 图标和表情符号，提升视觉体验

## 🎨 视觉标识

### 统计面板颜色
- 🔵 **蓝色** - 当前体重
- 🟢 **绿色** - 体重减少（总变化）
- 🔴 **红色** - 体重增加（总变化）
- 🟣 **紫色** - 记录数量
- 🟠 **橙色** - 最高体重
- 🟢 **青色** - 最低体重

### 记录列表标识
- 🔴 **红色 ↑** - 体重增加
- 🟢 **绿色 ↓** - 体重减少
- ⚪ **灰色 ●** - 体重持平
- 🔵 **蓝色 ●** - 首次记录

## 🚀 使用方法

1. 在"体重记录"标签页中输入体重值（例如：70.5）
2. 点击"添加记录"按钮或按回车键
3. 查看统计面板了解总体趋势
4. 浏览历史记录列表查看详细变化

## 💾 数据存储

所有体重记录保存在应用根目录的 `weight_records.json` 文件中。

## 🎯 统计说明

- **总变化**：显示从第一条记录到最新记录的体重变化
  - 如果总变化为正（增加），显示红色 ↑
  - 如果总变化为负（减少），显示绿色 ↓
  - 如果没有变化，显示灰色 ●

## 🎬 动画效果

- 添加新记录时，统计数据会有淡入淡出的过渡效果
- 列表更新时平滑刷新

## 📝 输入验证

- 必须输入有效的数字
- 体重必须大于 0
- 体重范围：20-300 kg
- 支持小数点后一位精度
